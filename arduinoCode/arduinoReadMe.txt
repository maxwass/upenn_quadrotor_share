This folder contains
1) the library that must be installed on a computer with an arduino ide
2) the code to be pushed to the arduino with the imu mounted
3) the pdf outlining the instructions of how to set up the arduino ide and install the library


The code to be pushed to the arduino: "AccAndGyrCalibration_2.ino"
description: outputs are gyro raw data and pose estimation in arrays of floats and uint_16's

The library "Invensense_ICM30630_eMD_ArduinoZero_1.pfdf"
description: contains files to support the function calls in the file above

The pdf "Tungsten_Software_User_Guide.pdf"
description: walks through how to install arduino ide and select correct port. Also goes through how to include library for use (pg 9)
